public final startCamera()

Start the camera. When the camera starts onCameraStarted() is called and the captured preview frames will start to be passed to the cameraDelegate.

stopCamera

Introduced in version

public final stopCamera()

Stop and release the camera. This method is called automatically in onPause() and onStop(). After this method is called the cameraDelegate will stop receiving the preview frames from the camera.
